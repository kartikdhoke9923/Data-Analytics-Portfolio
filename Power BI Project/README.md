This repository contains an interactive analytics solution built using **Power BI**, focused on analyzing the total number of electors in Mumbai City for the year 2024.

---

## Project Overview

This Power BI report helps understand elector distribution across various regions of Mumbai City.  
With the help of calculated measures, slicers, drill-downs, and KPI indicators, the dashboard enables users to explore elector trends and patterns effectively.

The goal of this project is to create a dynamic and visually clear representation of elector data to support data-driven insights.

---

## Tools & Technologies Used

- **Power BI Desktop**
- **Power Query** (Data cleaning & transformation)
- **DAX** (Custom measures and KPIs)
- **Excel / CSV** (Data source)
- **Git & GitHub** for version control
